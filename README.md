# dotfiles
Just browse the dotfiles themselves for nice configs, otherwise you can install them using the procedure below:

1. Get the dotfiles
```bash
git clone https://github.com/peteretelej/dotfiles.git 
cd dotfiles                                           
```
2. Run dotfiles installation script
```bash
./install.sh
```


# TIPS
